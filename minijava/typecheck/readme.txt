解释一下typecheck的bonus部分：
————————————————————————————
对各种变量的初始化
类变量：已初始化（与java一致）
数组：已初始化但不确定值（不能在编译阶段就保存一整个数组的数据）
函数的返回值：已初始化但不确定值

方法变量：未初始化
字面值：已初始化
—————————————————————————————
只要涉及到运算
如果操作数中有未初始化，报错，返回的变量设成未初始化
如果操作数全部初始化，返回一个已初始化的值
否则，返回一个已初始化但不确定值（new出来的默认值）

如果是函数操作，对其每个实参a检查
a如果是未初始化，报错，同时返回的变量设成未初始化
（如果全部初始化，也返回一个已初始化但不确定值）
否则，返回一个已初始化但不确定值（new出来的默认值）

涉及到赋值和return：
如果右侧是未初始化，报错，左侧变成已初始化但不确定值（先不变为未初始化）
如果右侧已初始化，则左侧也变为已初始化
否则，左侧变成初始化且不确定值
—————————————————————————————
a = new int[已初始化]：size=已初始化
a = new int[已初始化但不确定值]：size=已初始化但不确定值
a = new int[a未初始化]：报错，size=已初始化但不确定值（先不变为未初始化）
